Points Counter Help

Bot Runners:

--twitch

This argument tells the code to run a Twitch bot with the credentials given in the config, if the required credentials are not given, the program will exit stopping all bots.

--discord

This argument tells the code to run a Discord bot with the credentials given in the config, if the required credentials are not given, the program will exit stopping all bots.

--mixer

This argument tells the code to run a Mixer bot with the credentials given in the config, if the required credentials are not given, the program will exit stopping all bots.



Misc Utilites:

--test

Runs the tests for the various commands.

--compile-docs

Compiles the documentation for all the commands, this prevents the need for a web server to serve to documentation on.
