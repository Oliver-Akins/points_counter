Points Modification Commands

The !add is now !points add.
The !remove is now !points remove.

The syntax of these commands also changed, read the following pages to learn about the syntax change:
points add
points remove

Auto-Addition of Twitch Bits

The bot will now automatically add bits to a user based on if they included a target in the message with the bits. Whether or not the bot responds depends on a configuration value, for Starwoo, it is set to respond to the user sending the bits if it matches and attempts to add them, otherwise it will ignore the message.

Missed Twitch Bits in Discord

If a user sends a bit message in the chat without specifying a target, the bot may send a message into Discord with the donator and the total number of bits, indicating that they were not added automatically, this allows the mods to not waste time manually counting a large number of bits that are only 1 bit each.

Option Listing

To list options now, the command is !options list.
