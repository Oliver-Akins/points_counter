Stardew Valley Date Counter Help

Commands:

User Commands:

!lead
Will tell you who's winning the race with the most bits.

!ping
Checks if bot is online.

!help
Brings you to this page.

!top
Lists the top 3 characters and their respective bit amounts.

Moderator Commands:

!add [Donator]
Adds the given amount to the IGP on behalf of Donator

!remove [Chat User]
This command will remove the given "points" from the given "IGP", IF you specify the "Chat User" (the person who donated the bits) it will be added under their name, if this is not given, it will remove them from the anonymous user. ONLY USE THIS COMMAND IF YOU MAKE A MISTAKE WHILE ADDING.
